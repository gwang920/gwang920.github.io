## I. 주제 및 기획의도

- 주제 : JSP를 활용한 게시판 제작
- 기획의도 : Spring MVC 패턴과 MVC1 패턴의 차이 알아보기



## II. 기능 개요

- 회원가입, 로그인 기능
- 게시판 기능
- 댓글 기능
- 클라이언트 삭제 요청 수행 시, **isAvailable**을 통해 DB에 데이터 보존
